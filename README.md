# BASE_OF_ROBOT
this project is about a base for the previous robot arm project
the project contain an interface let you control the base through movments like forward,backward ,left,right and stop
this interface is connected with database save your movments 
this project contain a php page that connect between the database and the hardware through printing the last row in the database
i combine the interface of this project and the previous project "robot_arm"
the last thing i do that show the interface as mobile web view "iphonex"
